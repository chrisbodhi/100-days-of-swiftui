# 100 Days of SwiftUI

SwiftUI projects from [Paul Hudson's 100 Days of SwiftUI](https://www.hackingwithswift.com/100/swiftui) course. Worked through a solid chunk of the curriculum to build familiarity with the framework before moving on to other projects.

## Projects

| Project | What it does | Key concepts |
|---|---|---|
| **WeSplit** | Bill-splitting calculator with tip options | Forms, `@State`, `@FocusState`, currency formatting |
| **GuessTheFlag** | Flag identification game with animations | 3D transforms, `RadialGradient`, rotation/opacity animations |
| **RockPaperScissors** | 10-round game against a random opponent | Game logic, alerts, conditional UI |
| **ViewsAndModifiers** | Playground for custom modifiers and UIKit bridging | `ViewModifier`, `UIViewRepresentable`, PDFKit |
| **Animations** | Showcase of SwiftUI animation primitives | `DragGesture`, custom `AnyTransition`, staggered delays |
| **WordScramble** | Make words from a shuffled root word | File I/O, `UITextChecker`, list management |
| **BetterRest** | Sleep calculator backed by a CoreML model | CoreML integration, `DatePicker`, custom `Binding` |
| **Edutainment** | Configurable multiplication quiz with a timer | `Timer.publish`, game flow, focus management |
| **iExpense** | Expense tracker with Personal/Business categories | `@StateObject`, `Codable`, sheet presentation |
| **Moonshot** | Apollo mission explorer with astronaut bios | `NavigationSplitView`, JSON decoding, multi-column layout |
| **colorConv** | Converts colors between HEX, RGB, and HSL | String parsing, UIKit color bridging |
| **SinceEpoch** | Apple Watch face showing Unix epoch time | watchOS, `Timer.publish`, live updating |
| **FlagXR** | 3D flag viewer for Vision Pro | RealityKit, immersive spaces, `Model3D` |
| **NavMigration** | NavigationSplitView pattern demo | Selection bindings, detail view management |

## Environment

- Xcode 15+
- iOS 17+ / watchOS 10+ / visionOS 1+
- CoreML model included in BetterRest (`BetterRest.mlmodel`)
